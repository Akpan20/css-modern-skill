# Contributing to CSS Modern Reference & Layout Architect

## 🛠️ How to Contribute

1. Fork & clone the repo
2. Create a feature branch: `git checkout -b feat/prompt-name`
3. Add your prompt to `prompts/` or update `.claude-instructions.md`
4. Test outputs in Claude Chat using the response framework
5. Submit a PR with:
   - Clear description of the change
   - Example input/output
   - Browser support notes if applicable

## 📐 Prompt Guidelines

- Keep prompts under 150 words
- Always request: cascade impact, fallback strategy, support tags
- Use `${1:placeholder}` syntax for VS Code snippet compatibility

## 🔒 Code of Conduct

- Respect spec accuracy over opinion
- Cite W3C/MDN sources for bleeding-edge features
- Maintain progressive enhancement as default

Thank you for helping keep this skill production-ready!
